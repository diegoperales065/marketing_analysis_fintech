# 🤖 Agentes de Automatización — TFM Marketing Bancario

Este directorio contiene las **skills** (habilidades técnicas) y los **workflows** (flujos de trabajo) que automatizan y estandarizan los procesos del Trabajo Fin de Máster.

---

## 📁 Estructura

```
.agents/
├── README.md                 ← Este archivo
├── skills/                   ← Habilidades técnicas reutilizables
│   ├── pdf/                  ← Lectura y procesamiento de PDFs
│   ├── data-cleaning/        ← Limpieza y preparación de datos
│   ├── eda/                  ← Análisis Exploratorio de Datos
│   └── modeling/             ← Modelado predictivo y evaluación
└── workflows/                ← Flujos de trabajo paso a paso
    ├── setup-environment.md  ← Configuración del entorno
    └── full-pipeline.md      ← Pipeline completo del TFM
```

## 🧠 Filosofía

- **Skills**: Conocimiento técnico encapsulado. Cada skill enseña a un agente IA *cómo* ejecutar una tarea específica.
- **Workflows**: Secuencias ordenadas de pasos. Definen *qué* hacer y en *qué orden*.
